# 專案名稱
DisWork

## 介紹


## 功能
1. 討論版

2. 活動行事曆功能
- 串接Google行事曆
- 即時建立事件
- 查看未來行程

3. 新聞
- 自動更新每日頭條新聞

4. 會員系統
- 第三方登入，快速方便
- 劃分會員等級，解鎖付費功能

5. 即時聊天互動

## 專案初始化
#### 虛擬環境:
```
$ pip install poetry
$ poetry init -n
$ poetry shell
$ poetry install
```
## Node.js
```
$ install node
$ npm init -y
$ npm run dev
```
```
$ make server
```

## 使用技術
- 前端：Tailwindcss/Alpine.js/HTMX
- 後端：Python Django
- 資料庫：mySQL
- 第三方串接：Google Calendar/Google OAuth 2.0/藍新金流
- 部署：AWS S3
